# Цифровой дресс-код: фон, который выделяет вас

### Краткий обзор проекта

| Категория | Описание |
| :--- | :--- |
| **Проблематика** | Существующие решения сегментации видео в корпоративных системах (например, DION) работают некачественно: создают визуальные артефакты (размытые, "зубчатые" края), нестабильны при движении и плохом освещении. Это подрывает профессиональный имидж сотрудников и бренда T1 в целом. Отсутствует централизованная система корпоративных фонов. |
| **Потенциальные пользователи** | **Внутренние:** сотрудники T1 всех уровней (от рядовых до топ-менеджмента), HR-специалисты (для онбординга), отделы продаж и маркетинга.<br>**Внешние:** клиенты и партнеры T1, взаимодействующие с сотрудниками компании через видеосвязь. |
| **Решение** | Высококачественный и производительный **ML-модуль сегментации**, работающий в браузере. Он обеспечивает стабильное выделение контура человека даже в сложных условиях. На его основе реализован **генератор персонализированных корпоративных фонов** с динамической подстановкой данных сотрудника (имя, должность), соблюдением принципов контрастности и приватности. |
| **Конкуренты** | **Внутренний конкурент:** Текущая реализация сегментации в сервисе DION.  <br>**Внешние конкуренты:** Встроенные функции замены фона в Zoom, Microsoft Teams, Google Meet, которые часто имеют схожие проблемы с качеством и не предлагают глубокой корпоративной кастомизации. |
| **Уникальность решения** | **1. Баланс качества и производительности:** Достижение высокой точности сегментации при строгих ограничениях на нагрузку CPU/GPU (≤30%/40%) и задержку (≤12 мс), что обеспечивает плавную работу на средних ноутбуках. <br> **2. Глубокая персонализация:** Не просто замена фона, а создание "цифрового бейджа" сотрудника с динамически обновляемыми данными и гибкими настройками приватности. <br> **3. Работа в браузере:** Решение полностью выполняется на стороне клиента с помощью JavaScript, что не требует серверных мощностей и упрощает интеграцию. |
